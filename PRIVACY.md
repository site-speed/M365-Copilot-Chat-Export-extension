# Privacy Policy — M365 Copilot Chat Conversation Exporter Extension

Last updated: 2026-05-14

## Summary

M365 Copilot Chat Conversation Exporter Extension helps you export the Microsoft 365 Copilot Chat conversation that is already open in your browser into local Markdown and JSON Markdown files.

The extension is designed for local export. It does not sell data, broker data, run analytics, inject advertising, or transmit exported conversation contents to any third-party service.

## Data the extension accesses

When you use the extension on supported Microsoft 365 Copilot Chat pages, it may access:

- the current conversation title and conversation ID;
- user prompts and Copilot responses in the selected conversation;
- citations, links, source provenance, reasoning/process summaries, plugin/tool provenance, and uploaded filename metadata that are part of the conversation record;
- browser session/authentication material already present in the Microsoft 365 page context, only as needed to request the current conversation from Microsoft 365 services.

## How data is used

The extension uses this data only to identify the current conversation, request conversation data from Microsoft 365 services that your signed-in browser session can already access, and generate local export files.

## Data storage and transfer

The extension does not operate a backend service and does not upload exported data to the extension author.

Exported files are downloaded by your browser to a location you choose or to your browser's configured downloads folder. Those files may contain sensitive work or personal data and should be handled according to your organisation's policies.

The extension communicates with Microsoft 365 and Substrate service endpoints only as required to retrieve the current conversation in the context of your authenticated Microsoft 365 browser session.

## Permissions

The extension requests only the permissions needed to identify supported Microsoft 365 Copilot Chat pages, communicate between the extension popup/content/page bridge, and download generated export files.

## Third parties

The extension does not share exported conversation content with third parties.
